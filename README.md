# QuantaNow - AI Agency Website

A modern, high-end website for QuantaNow, an AI-powered software development agency.

## 🚀 Features

- **Modern Design**: Clean, asymmetric layouts with premium aesthetics
- **AI Agency Focused**: Tailored content for AI development services
- **Responsive**: Optimized for all devices and screen sizes
- **Performance**: Fast loading with optimized animations
- **Interactive**: Smooth scrolling, hover effects, and dynamic content
- **Professional**: Enterprise-grade design comparable to top SaaS companies

## 🚀 Getting Started

### Option 1: Open Locally
1. Navigate to the project folder
2. Double-click `index.html` to open in your browser

### Option 2: Local Server (Recommended)
1. Install a local server (e.g., Live Server extension for VS Code)
2. Right-click `index.html` and select "Open with Live Server"
3. Or use Python: `python -m http.server 8000`
4. Visit `http://localhost:8000`

### Option 3: Development Server
```bash
# Using Node.js serve package
npx serve .

# Using Python
python -m http.server 8000

# Using PHP
php -S localhost:8000
```

## 🔧 Customization

### Colors
Modify CSS custom properties in `styles.css`:
```css
:root {
  --primary-black: #0A0A0A;
  --accent-green: #00E887;
  --secondary-blue: #2563EB;
}
```

### Content
Update text content directly in `index.html` or create a CMS integration.

### Animations
Adjust timing and effects in `script.js` and CSS transition properties.
